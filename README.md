# frontend-gulp

Gulp 3 script for Front-end developers

Work commands:
*npm install* - install npm
*gulp* - run gulp
*gulp clean* - clean dist folder

If you work with WordPress. You can uncomment urls for streaming css, js, and images. Please don't forget a change theme name.

Include your html and js files as modules. You can find examples inside *index.html* and *js/scrips.js*
html:
//= include/header.html
js:
//= modal.js

Integrated modules:
- jQuery
- Custom Modal script
- Slick slider
